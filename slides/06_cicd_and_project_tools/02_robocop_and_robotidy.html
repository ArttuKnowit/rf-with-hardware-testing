Robocop and Robotidy

Robocop and Robotidy are tools for linting and formatting robot framework test cases. Robocop is a linter that checks the code against a set of rules and provides feedback on any issues found. Robotidy is a formatter that automatically formats the code according to a set of rules. Both tools can be used to improve the readability and maintainability of robot framework test cases, as well as to enforce coding standards and best practices. They can be integrated into the development workflow, such as in a CI/CD pipeline, to ensure that the code is always in a good state.
