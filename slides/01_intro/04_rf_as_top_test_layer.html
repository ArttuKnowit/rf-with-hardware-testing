Robot Framework - Top Test Layer

To best utilise Robot Framework as a test framework, each test case should be built with top-level keywords in Robot Framework, and all technical details should be left for the adaption layer. Test cases should consist of keywords that describe in high level what is happening in the test case. These keywords are like functions and should be designed and constructed for re-usability. The reason to use Robot Framework relies in built-in reporting tools and capabilities.

Consider the examples on the right:

  • Test Satellite GPS Function is an example of a test case that has too little business-level description. It is not clear what the test case is doing and what the expected result is.
  • Test Satellite Lights is an example of a test case that is too technical and bloated with technical level keywords. It is not clear what the test case is doing and what the expected result is.
  • Test Satellite Temperature Sensor is an example of a test case that strikes a balance. It has a clear business-level description and it is not too technical. It is clear what the test case is doing and what the expected result is. It also benefits from the correct usage of test case settings.

Test Satellite Lights
    Connect To Satellite    voyager
    ${light_1_status}=    Get Light Status    light_1
    Should Be Equal    ${light_1_status}    off
    Turn On Light    light_1
    ${light_1_status}=    Get Light Status    light_1
    Should Be Equal    ${light_1_status}    on
    ${light_2_status}=    Get Light Status    light_2
    Should Be Equal    ${light_2_status}    off
    Turn On Light    light_2
    ${light_2_status}=    Get Light Status    light_2
    Should Be Equal    ${light_2_status}    on
    Disconnect From Satellite

Test Satellite Temperature Sensor
    [Documentation]    A temperature sensor test, 
    ...   should be run at room temperature
    [Setup]    Connect To Satellite    voyager
    ${temperature}=    Get Temperature From Sensor
    ...    temperature_sensor_1
    Should Be Less Than    ${temperature}    25
    Should Be Greater Than    ${temperature}    15
    [Teardown]    Disconnect From Satellite
