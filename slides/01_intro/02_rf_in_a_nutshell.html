Robot Framework in a nutshell

Robot Framework is a great top layer for maintainable test automation. For an experienced python user, it brings very little to the table in terms of technical power. Its strength lies in automatically and implicitly handling many aspects of test execution and reporting, as well as being very modular and having a wide open-source ecosystem.

On the right is an example of a simple test case file in Robot Framework. This file wraps more complicated and technical code usually made with python into a individual test cases formed of keywords.

A test case consists of a name, settings and keyword calls. 

*** Settings ***
Library    CustomLibrary.py

*** Variables ***
${SERIAL_PORT}    1234

*** Test Cases ***
Test LED Timer
    [Setup]   Connect To Device    ${SERIAL_PORT}
    Turn On LED
    ${led_status}=    Get LED Status
    Should Be Equal    ${led_status}    ON
    Sleep    5s
    ${led_status}=    Get LED Status
    Should Be Equal    ${led_status}    OFF
    [Teardown]    Disconnect From All Devices
