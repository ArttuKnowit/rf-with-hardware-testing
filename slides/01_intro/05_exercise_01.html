Exercise 01 - Hello World

  1. Create a new file called hello_world.robot under suites/examples/
  2. Create a section for variables and define a variable called ${NAME} with a value of your preference
  3. Create a section for test cases
  4. Create a new test case called Greeting test
  5. Within the test case body, call the keyword Log To Console with the argument Hello, ${NAME}
