Course agenda

This is a two-day course in which we will try to explore the following topics regarding Robot Framework:

  1. Robot Framework as a testing framework
  2. Basic and intermediate syntax
  3. Creating tests, test suites and keywords with Robot Framework
  4. Running tests and generating reports
  5. Best practices for test automation with Robot Framework
  6. Extending Robot Framework with Python
  7. Linting and code quality
  8. Dockerizing tests
