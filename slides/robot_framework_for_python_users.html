Robot Framework For Python Users

Robot Framework is a great way to use a generally defined wrapper for tests to produce maintainable test automation. For an experienced python user, it brings very little to the table in terms of technical power. Its strength lies in automatically and implicitly handling many aspects of test execution and reporting, as well as being very modular and easily extendable and having a wide open-source ecosystem.

On the right is an example of a simple test case file in Robot Framework. This file wraps more complicated and technical code usually made with python into a individual test cases formed of keywords.


*** Settings ***
Library    CustomLibrary.py

*** Variables ***
${SERIAL_PORT}


*** Test Cases ***
Test LED Timer
    [Setup]   Connect To Device    ${SERIAL_PORT}
