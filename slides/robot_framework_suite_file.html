Robot Framework - File Sections

Suite files have a specific structure with different sections. The possible sections in a suite file are:

  • Settings
    • Used to import libraries, resource files and variables, to set suite level setup and teardown tasks, and to define other settings related to the test suite.
  • Variables
    • Used to define static variables on suite level scope.
  • Test Cases
    • Used to define test cases, which are the main building blocks of test suites.
  • Keywords
    • Used to define user keywords, which are reusable building blocks that can be called from test cases or other keywords.
  • Comments
    • Used to write comments that are ignored by Robot Framework and are only for documentation purposes.

*** Settings ***
Library    CustomLibrary.py

*** Variables ***
${SERIAL_PORT}    1234

*** Test Cases ***
Test LED Timer
    [Setup]   Connect To Device    ${SERIAL_PORT}
    Turn On LED
    ${led_status}=    Get LED Status
    Should Be Equal    ${led_status}    ON
    Sleep    5s
    ${led_status}=    Get LED Status
    Should Be Equal    ${led_status}    OFF
    [Teardown]    Disconnect From All Devices
