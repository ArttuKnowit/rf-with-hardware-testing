Custom Libraries

Custom libraries are a powerful and almost a standard way to extending Robot Framework's functionality, or to provide it proper functionality to begin with. While they don't need to be strictly written in python, it is the most practical and viable way. The Robot Framework's own classes and methods are designed for building custom libraries in python, and the documentation and community support it best.

Custom libraries are imported in the settings section of a suite file just like any other library, and they can be used just like any other library. While it is possible to write individual functions in a custom library and import them into Robot Framework, the standard way is to create a Library Class, which is imported as an object into the test suite. The methods of the class are then used as keywords.

There is an example to the right, and you may take a look at ArduinoLib.py found in the libraries directory.

Writing a library is straight forward. Import dependencies as you would normally and create a class object that shares the name with the file itself but without the .py extension (so for example, in a file called ArduinoLib.py you should create a class called ArduinoLib). Import the keyword and library decorators from robot.api.deco, and decorate the class with @library and the methods you want to use as keywords with @keyword. The method names will be the keyword names, but you can also specify a custom name for the keyword in the decorator if you want to.

The library is imported with the file reference, so taking into account the relative path. However, if a more structured python package is created with it's own wheel, it can be isntalled to the environment and imported just like any other library.

CustomLibrary.py 
from robot.api.deco import library, keyword
import serial

@library
class CustomLibrary:
    def __init__(self):
        self.connection = None

    @keyword
    def connect_to_device(self, serial_port,
                          baudrate=9600, timeout=2):
        self.connection = serial.Serial(
            port=serial_port,
            baudrate=baudrate,
            timeout=timeout
        )
