Exercise 02 - System Under Test

  1. Use your Arduino kit and the Arduino projects book to assemble the circuit found in project 02 (page 34). You may also use all the time you need to get familiar with the Arduino.
  2. Connect the Arduino to your computer with a USB cable. The board itself should have a light turn on indicating it's powered, but the three LED's should remain off.
  3. Ensure you have the arduino-cli installed and configured using `arduino-cli version`. Ask the instructor if you do not yet have it installed on your device.
  4. Compile the firmware using the command arduino-cli compile --fqbn arduino:avr:uno firmware/project_02
  5. Identify the port to which the Arduino is connected. You can use the command arduino-cli board list to find the correct port.
  6. Flash the firmware to the Arduino using the command arduino-cli upload -p [PORT] --fqbn arduino:avr:uno firmware/project_02, replacing [PORT] with the correct port.
  7. Verify that the firmware is running correctly by observing the LED behavior on the Arduino as you press the button. At this point the Red LED's should light up briefly as you press the button, until the green light turns on again.
  8. Try flashing the firmware using the project_02_custom directory instead. Press the button to observe the difference.
