Control Structures

Robot Framework control structures all function very similarly to python control structures. Syntax-wise the largest difference is that control structures are enclosed with an END statement. Robot Framework has

  • FOR and WHILE loops
  • IF/ELSE/ELSE IF statements
  • TRY/EXCEPT for error handling
Most complex nested control structures are usually embedded within the python code. Control structures in Robot Framework are used in cases where the logic is very clear and descriptive, and integral to the overall flow of the test case.

Test LED Toggling
    [Setup]   Connect To Device   ${SERIAL_PORT}
    ${LEDs}=    Get LEDs
    FOR    ${led}    IN    @{LEDs}
        ${led_status}=    Get LED Status    ${led}
        IF    ${led_status}
            Turn Off LED    ${led}
        ELSE
            Turn On LED    ${led}
        END
    END
    [Teardown]    Disconnect From All Devices
