Environment Variables

Environment variables are used to access variables set in the operating system before the test execution. This allows easily accessing all OS environment variables just like one would from the command line. Environment variables can also be created or deleted with keywords Set Environment Variable and Delete Environment Variable which are available in the built-in OperatingSystem library. Environment variables are always global.


*** Test Cases ***
Check Environment Variables
    Log    Current user is %{USER}
    Run    %{JAVA_HOME}/javac

Use Environment Variable With Default Value
    Set Environment Variable    %{APPLICATION_PORT=8080}
