Robot Framework - Settings

The settings section of a suite file is used to define the following:

  • The documentation for the suite using Documentation setting
  • Metadata for the suite using Metadata setting
  • Libraries, resource files, and variables needed for the test cases in the suite, using Library, Resource, and Variables settings
  • Setup and teardown tasks for the whole suite using Suite Setup and Suite Teardown
  • Test case level setup and teardown tasks using Test Setup and Test Teardown that are common to all test cases in the suite. They take place if test cases do not have their own setup or teardown set in their own settings.
  • Tags shared by each test case in the suite using Test Tags setting
  • Test Template for the suite using Test Template setting, explained more thoroughly later.

*** Settings ***
Documentation    This is the documentation for the test suite.
Metadata         Author: John Doe
Library          CustomLibrary.py
Resource         arduino_led_control.resource
Variables        variables.py
Suite Setup      Connect To Device    ${SERIAL_PORT}
Suite Teardown   Disconnect From All Devices
Test Setup       Reset Device State
Test Teardown    Flash All LEDs
Test Tags        critical    led_control
