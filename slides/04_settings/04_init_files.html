The __init__.robot File

To set settings for the suite that is a directory that contains multiple test case fales, an __init__.robot file can be used. In that file

*** Settings ***
Library    CustomLibrary.py
Resource    arduino_led_control.resource
Variables    variables.py
Suite Setup    Connect To Device    ${SERIAL_PORT}
Suite Teardown    Disconnect From All Devices

*** Keywords ***
Suite Test Case Setup Task
    [Documentation]    Common setup task for all sub suites in this suite.
    [Arguments]     ${port}
    Connect To Device    ${port}

Suite Test Case Teardown Task
    [Documentation]    Common teardown task for all sub suites in this suite.
    Disconnect From All Devices
